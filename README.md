About the Project

Digital Tools is a responsive and modern web application designed as a digital marketplace. It includes a dynamic product listing system, an efficient cart management feature, and a polished UI to enhance digital productivity.

Technologies Used
Framework : React.js
Styling : Tailwind CSS & DaisyUI
Icons : Lucide React
Notifications : React-Toastify
Data Management : JSON-based product structure
JavaScript (ES6+)


React + Vite
This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.
Currently, two official plugins are available:

@vitejs/plugin-react uses Oxc
@vitejs/plugin-react-swc uses SWC 

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the ESLint configuration

Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and [`typescript-eslint`](https://typescript-eslint.io) in your project.
