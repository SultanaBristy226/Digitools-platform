About the Project

Digital Tools is a responsive and modern web application designed as a digital marketplace. It includes a dynamic product listing system, an efficient cart management feature, and a polished UI to enhance digital productivity.

Technologies Used
Framework : React.js
Styling : Tailwind CSS & DaisyUI
Icons : Lucide React
Notifications : React-Toastify
Data Management : JSON-based product structure
JavaScript (ES6+)
